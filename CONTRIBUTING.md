# Contributing to Multi-Agent Book Writer & Editor

Thank you for your interest in contributing!

This project implements a multi-agent architecture for generating fictional stories
using a Planner → Worker → Evaluator pipeline.

## How to Contribute

### 1. Fork the Repository
Create your own copy and clone it locally.

### 2. Project Structure

project/
  agents/
  tools/
  memory/
  core/
  main_agent.py
  app.py
  requirements.txt

### 3. Contribution Ideas
- Add new agent tools (rewriters, grammar checkers)
- Improve story continuity checks
- Optimize evaluator scoring system
- Add new memory strategies

### 4. Coding Standards
- Follow Python PEP8
- Functions should remain small and modular
- Logging is mandatory for all agent outputs
- Avoid adding heavy libraries unless necessary

### 5. Pull Requests
Before submitting:
- Ensure all tests pass (if tests exist)
- Provide a clear explanation of your changes

We welcome all contributions — from bug fixes to new features!
